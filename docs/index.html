LuaSynth

A modular synthesiser built in pure C with Lua 5.4 as the scripting engine. The project is a learning exercise in two areas simultaneously:

  • DSP fundamentals — oscillators, filters, envelopes, delay lines
  • Embedding Lua in C — full userdata, metatables, callbacks, thread safety

Everything is deliberately simple and thoroughly commented. Read the C source alongside the Lua examples to understand how scriptable applications are structured.

Quick Start

macOS (Homebrew)

brew install sdl2 lua
git clone … && cd luasynth
make run-01          # plays 440 Hz sine tone

Linux (apt)

sudo apt install libsdl2-dev lua5.4 liblua5.4-dev
make run-01

The simplest patch

-- Play a 440 Hz sine tone
local osc = Osc.new("sine", 440.0)
Output.set(osc)
Run any example: ./luasynth scripts/examples/01_sine_tone.lua — or use make run-01 through make run-06.

Architecture

Four interlocking layers:

Lua Script ──→ creates modules, sets parameters, calls Output.set()
↓ Lua C API (full userdata + metatables)
Lua Binding Layer ──→ wraps C structs as Lua objects with methods
↓ SDL_LockAudioDevice / _Atomic for thread safety
Signal Graph ──→ DAG of modules; output pulls from upstream recursively
↓ SDL audio callback (separate thread)
SDL2 Audio Engine ──→ calls signal_graph_process() every ~11.6 ms

Pull Model

The audio callback asks the output module for samples. The output module asks its upstream inputs, which ask their inputs, and so on — recursively through the DAG. No explicit traversal order is needed; the recursion follows the graph edges.

Each module's process(self, out_buf, num_frames, sample_rate) function fills a stereo interleaved float buffer. Pre-allocated scratch buffers on each module avoid malloc() in the real-time audio thread.

Thread Safety

SituationMechanism
Lua sets osc frequency / filter cutoff SDL_LockAudioDevice / SDL_UnlockAudioDevice
Lua calls env:trigger() atomic_store(&data->triggered, 1) (C11 _Atomic)
Clock beat fires (audio thread) → call Lua Push event to SPSC ring buffer; main loop calls Lua callback
Rule: The Lua VM must only be called from the main thread. Never call Lua from the SDL audio callback.

Module Reference

Osc — Oscillator

Generates periodic audio waveforms using a phase accumulator. Phase advances by frequency / sample_rate per sample and wraps at 1.0, producing exactly frequency cycles per second regardless of sample rate.

Waveforms are naive (no PolyBLEP anti-aliasing) — aliasing occurs at higher frequencies, which is intentional for this educational project.

Constructor

ArgumentTypeRangeDescription
waveformstring"sine" "square" "saw" "triangle"Waveform shape
frequencynumber0.1 – 20000 HzInitial frequency in Hz
local osc = Osc.new("sine", 440.0)

Methods

MethodArgumentsReturnsDescription
setFrequency(hz)hz number — HznilSet oscillator frequency. Thread-safe (SDL lock).
setAmplitude(amp)amp number — [0,1]nilSet output gain.
setWaveform(name)name stringnilChange waveform at runtime.
connect(src)src modulenilAdd src as an audio input (additive mix).
connectMod(dst, param, depth)dst module, param string, depth numbernilWire this osc as a control-rate modulator of dst.param.

Example

-- Sawtooth at A2 with 50% amplitude
local osc = Osc.new("saw", 110.0)
osc:setAmplitude(0.5)
Output.set(osc)

-- Change to square wave after 1 second
SDL.delay(1000)
osc:setWaveform("square")

Noise — White Noise Source

Generates white noise: random samples from [−1, 1] uniformly distributed. White noise contains equal power at all frequencies — ideal as a filter input for drum synthesis and texture.

Constructor

ArgumentTypeDescription
none—Noise.new() takes no arguments
local n = Noise.new()

Methods

MethodArgumentsReturnsDescription
setAmplitude(amp)amp number — [0,1]nilSet output gain.
connect(src)src modulenilAdd an audio input (not typical for noise, but supported).

Example

-- White noise through a tight bandpass — snare drum texture
local noise = Noise.new()
noise:setAmplitude(0.3)
local flt = Filter.new("svf")
flt:setInput(noise)
flt:setCutoff(3000)
flt:setResonance(3.0)
flt:setMode("bandpass")
Output.set(flt)

Filter — State-Variable Filter & Moog Ladder

Two filter topologies:

  • SVF (Chamberlin State-Variable Filter) — simultaneous lowpass / bandpass / highpass outputs. Very stable across the cutoff range. Update equations per sample: low += f*band; high = in - low - q*band; band += f*high
  • Moog — 4-stage lowpass ladder with resonance feedback. Models transistor saturation with fast_tanhf(). Classic warm sound; self-oscillates at resonance ≈ 1.

Constructor

ArgumentTypeValuesDescription
typestring"svf" "moog"Filter topology
local flt = Filter.new("svf")

Methods

MethodArgumentsReturnsDescription
setInput(src)src modulenilConnect upstream audio source.
setCutoff(hz)hz number — [20, 20000]nilSet cutoff frequency. Thread-safe.
setResonance(res)res number — SVF [0,4], Moog [0,1]nilSet resonance/Q.
setMode(mode)mode string — "lowpass" "bandpass" "highpass"nilSVF output tap (Moog is always lowpass).
connectMod(src, param, depth)src module, param "cutoff", depth numbernilWire an LFO as a cutoff modulator.

Example

local osc = Osc.new("saw", 220)
local flt = Filter.new("moog")
flt:setInput(osc)
flt:setCutoff(1200)
flt:setResonance(0.8)
Output.set(flt)

LFO — Low-Frequency Oscillator

Structurally identical to Osc but intended for control-rate use (0.01–20 Hz). Connect it to a target module's parameter with connectMod() to create periodic modulation.

The LFO's process() output is in [−1, 1] and is scaled by the depth value specified in connectMod().

Constructor

ArgumentTypeRangeDescription
waveformstring"sine" "square" "saw" "triangle"LFO shape
ratenumber0.01 – 20 HzLFO rate
local lfo = LFO.new("sine", 0.5)

Methods

MethodArgumentsReturnsDescription
setFrequency(hz)hz numbernilUpdate LFO rate.
setDepth(d)d number — [0,1]nilScale LFO output amplitude.
setWaveform(name)name stringnilChange LFO shape.
connectMod(dst, param, depth)dst module, param string, depth numbernilWire this LFO as a modulator. depth is in the units of the target parameter (Hz for frequency/cutoff, etc.).

Example

local flt = Filter.new("svf")
flt:setInput(Osc.new("saw", 220))
flt:setCutoff(800)

-- LFO sweeps cutoff ±600 Hz at 0.5 Hz
local lfo = LFO.new("sine", 0.5)
lfo:connectMod(flt, "cutoff", 600)

Output.set(flt)

Envelope — ADSR Envelope Generator

Shapes the amplitude of its input signal through four stages:

Level
A
D
S
R
──────────────────────────── Time →

trigger() starts the Attack stage. release() starts the Release stage. Both use C11 _Atomic stores — safe to call from any thread, no SDL lock needed.

Constructor

ArgumentTypeRangeDescription
attacknumber0.001 – 10 sAttack time in seconds
decaynumber0.001 – 10 sDecay time in seconds
sustainnumber0 – 1Sustain level
releasenumber0.001 – 10 sRelease time in seconds
local env = Envelope.new(0.01, 0.1, 0.7, 0.3)

Methods

MethodArgumentsReturnsDescription
setInput(src)src modulenilConnect upstream audio to shape.
trigger()—nilStart attack. Atomic — thread-safe.
release()—nilStart release. Atomic — thread-safe.
setAttack(s)s number secondsnilUpdate attack time.
setDecay(s)s number secondsnilUpdate decay time.
setSustain(level)level number [0,1]nilUpdate sustain level.
setRelease(s)s number secondsnilUpdate release time.

Example

local osc = Osc.new("sine", 440)
local env = Envelope.new(0.005, 0.1, 0.5, 0.2)
env:setInput(osc)
Output.set(env)

-- Pluck three notes
for i = 1, 3 do
    env:trigger()
    SDL.delay(400)
end

Delay — Circular Buffer Delay Line

Stores audio in a pre-allocated circular buffer and plays it back after a configurable delay time. Feedback creates echo trails; wet/dry mixing controls the balance between direct and delayed signal.

Buffer is pre-allocated at delay_create() time for DELAY_MAX_TIME_SECONDS = 2.0 seconds — no allocation in the audio callback.

Constructor

ArgumentTypeRangeDescription
delay_timenumber0.001 – 2.0 sInitial delay time in seconds
feedbacknumber0 – 0.99Feedback gain (0 = one echo, 0.99 = long tail)
local dly = Delay.new(0.375, 0.55)

Methods

MethodArgumentsReturnsDescription
setInput(src)src modulenilConnect upstream audio source.
setDelayTime(s)s number [0.001, 2.0]nilUpdate delay time. Thread-safe.
setFeedback(f)f number [0, 0.99]nilUpdate feedback amount.
setWet(w)w number [0,1]nilDelayed signal mix level.
setDry(d)d number [0,1]nilDirect signal mix level.

Example

local osc = Osc.new("sine", 440)
local dly = Delay.new(0.4, 0.6)
dly:setInput(osc)
dly:setWet(0.5)
dly:setDry(1.0)
Output.set(dly)

Clock — BPM Tempo Clock

Generates beat events at a configurable tempo. The clock operates on the audio thread (using a phase accumulator identical to the oscillator), pushing RingEvent values into a lock-free SPSC ring buffer each beat. The main thread drains the ring buffer and calls the registered Lua callback.

The clock produces no audio — it is a control-only module. It outputs silence and is wired into the output module's input list so it gets processed each frame.

Constructor

ArgumentTypeRangeDescription
bpmnumber1 – 300Beats per minute
local clk = Clock.new(120)

Methods

MethodArgumentsReturnsDescription
start()—nilBegin generating beats. Also wires clock into output module's inputs.
stop()—nilPause the clock.
setBPM(bpm)bpm numbernilChange tempo. Thread-safe.
onBeat(fn)fn function(beat)nilRegister a Lua callback called each beat with the beat number.

Example

local notes = { 60, 64, 67, 72 }
local osc = Osc.new("sine", midi_to_freq(60))
Output.set(osc)

local clk = Clock.new(120)
clk:onBeat(function(beat)
    local note = notes[((beat - 1) % #notes) + 1]
    osc:setFrequency(midi_to_freq(note))
end)
clk:start()

Global Functions

The global audio output sink. Exactly one module routes audio to the speakers.

FunctionArgumentsReturnsDescription

midi_to_freq(note)

Converts a MIDI note number to a frequency in Hz using equal temperament: freq = 440 × 2^((note − 69) / 12)

ArgumentTypeRangeDescription
notenumber0 – 127MIDI note number (69 = A4 = 440 Hz)
midi_to_freq(69)   -- → 440.0 Hz  (A4)
midi_to_freq(60)   -- → 261.6 Hz  (C4 middle C)
midi_to_freq(21)   -- → 27.5 Hz   (A0 lowest piano key)

SDL.delay(ms)

Pause script execution for ms milliseconds. The SDL audio thread continues running during the delay — use this in scripts for timed events without blocking audio. 

SDL.delay(1000)   -- pause 1 second; audio keeps playing

Composition Guide

LuaSynth patches are built by creating modules and connecting them. Here are the common patterns.

Pattern 1: Simple Signal Chain

The most basic pattern: source → processing → output.

-- Source → Filter → Output
local osc = Osc.new("saw", 220)
local flt = Filter.new("svf")
flt:setInput(osc)
flt:setCutoff(1000)
Output.set(flt)

Pattern 2: LFO Modulation

An LFO modulates a parameter (cutoff, frequency) by being wired as a control-rate input. The depth argument is in the units of the target parameter. 

-- LFO sweeps filter cutoff ±800 Hz at 0.5 Hz
local lfo = LFO.new("sine", 0.5)
lfo:connectMod(flt, "cutoff", 800)

-- LFO adds vibrato to oscillator pitch (±5 Hz)
local vib = LFO.new("sine", 5)
vib:connectMod(osc, "frequency", 5)

Pattern 3: Envelope Shaping

Envelopes shape the amplitude of a signal. Trigger them from a clock callback for rhythmic patterns. 

local env = Envelope.new(0.005, 0.1, 0.3, 0.2)
env:setInput(osc)

local clk = Clock.new(120)
clk:onBeat(function(beat)
    env:trigger()   -- atomic — safe from this callback
end)
clk:start()
Output.set(env)

Pattern 4: Additive Mixing

Connect multiple sources to the same module using connect(). The receiving module sums all its audio inputs. 

-- Two oscillators mixed together
local osc1 = Osc.new("sine", 440)
local osc2 = Osc.new("sine", 440 * 1.003)  -- slightly detuned
osc1:setAmplitude(0.4)
osc2:setAmplitude(0.4)
osc1:connect(osc2)   -- mix osc2 into osc1's output
Output.set(osc1)

Pattern 5: Full Voice (Oscillator + Filter + Envelope + Delay)

-- Complete synthesiser voice
local osc = Osc.new("saw", midi_to_freq(60))
osc:setAmplitude(0.5)

local flt = Filter.new("moog")
flt:setInput(osc)
flt:setCutoff(1200)
flt:setResonance(0.5)

local lfo = LFO.new("sine", 0.3)
lfo:connectMod(flt, "cutoff", 800)

local env = Envelope.new(0.01, 0.2, 0.5, 0.4)
env:setInput(flt)

local dly = Delay.new(0.3, 0.4)
dly:setInput(env)
dly:setWet(0.4)
dly:setDry(1.0)

Output.set(dly)

Build Guide

macOS (Homebrew)

# Install dependencies
brew install sdl2 lua

# Build
make

# Run examples
make run-01     # 440 Hz sine tone
make run-02     # filtered sawtooth with cutoff sweep
make run-03     # LFO modulation
make run-04     # rhythmic melody with envelopes
make run-05     # delay echo
make run-06     # full patch

Linux (apt — Debian/Ubuntu)

# Install dependencies
sudo apt install libsdl2-dev lua5.4 liblua5.4-dev

# Build and run
make run-01

Linux (dnf — Fedora/RHEL)

sudo dnf install SDL2-devel lua-devel
make run-01

Custom script

# Run any Lua script directly
./luasynth my_patch.lua
