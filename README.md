# luasynth
"vibe coded" app for coding synths with lua + sdl2. 

I wanted to learn more about building DSP/synth apps at a low level AND embedding lua in applications, so I had the AI tutor build out this basic framework in a way where all the code is documented such that I could pick al apart and learn how these two technologies work better. CLAUDE.md contains the prompts used to build this. Maybe this tool will be useful to someone else.
